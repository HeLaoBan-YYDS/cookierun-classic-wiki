## 2、网站主题色

### 结论

**默认选择亮色主题。**

CookieRun Classic 的视觉重点是焦糖色饼干、黄色果冻、红色糖果和明亮的蓝色点缀。亮色背景更符合休闲、可爱、糖果世界的游戏气质，也更适合展示色彩丰富的 Cookie 角色图。

深色主题可以作为用户切换选项，但不建议作为默认主题。

```css
/* 导航页主题色 - 亮色主题 */
--nav-theme: 24 94% 52%;          /* cookie orange，接近 #f36b13 */
--nav-theme-light: 40 100% 61%;   /* jelly gold，接近 #ffb838 */

/* 导航页主题色 - 暗色主题 */
--nav-theme: 24 96% 58%;          /* brighter cookie orange */
--nav-theme-light: 42 100% 68%;   /* brighter jelly yellow */
```

建议整体页面搭配：

```css
/* 亮色背景建议 */
--background: 42 60% 97%;
--foreground: 24 35% 16%;
--card: 0 0% 100%;
--border: 31 45% 86%;
--accent-red: 3 82% 57%;
--accent-blue: 202 85% 53%;
--accent-brown: 24 47% 35%;

/* 暗色背景建议 */
--background: 24 28% 9%;
--foreground: 42 50% 94%;
--card: 24 24% 13%;
--border: 24 22% 23%;
--accent-red: 3 88% 64%;
--accent-blue: 202 90% 64%;
--accent-brown: 27 58% 55%;
```

## 3、多语言

### 结论：语言优先级

1. **English**
2. **Thai**
3. **Korean**
4. **Japanese**

> 当前全球版商店标注的客户端语言只有英语。这里的多语言建议主要针对攻略站 SEO 和当地玩家搜索需求，不代表游戏客户端已经支持全部语言。

| 优先级 | 语言       | 本地化主题名                                | 推荐 URL | 分析                                                                  |
| --: | -------- | ------------------------------------- | ------ | ------------------------------------------------------------------- |
|   1 | English  | CookieRun Classic                     | `/en`  | 全球版当前主要官方语言，英文官方 X、Instagram、帮助中心和商店资料最完整，应作为主站语言。                  |
|   2 | Thai     | CookieRun Classic / คุกกี้รัน คลาสสิก | `/th`  | 泰国是旧 LINE Cookie Run 的核心市场之一，全球版上线后官方账号宣布登上泰国 App Store 第 1，当前热度很强。 |
|   3 | Korean   | 쿠키런 클래식                               | `/ko`  | 游戏源自韩国 Kakao 版，在韩国拥有长期玩家基础、官方韩语账号和大量原版攻略需求。                         |
|   4 | Japanese | クッキーラン：クラシック                          | `/ja`  | 日本区上线初期进入 Racing 分类前列，CookieRun IP 具有稳定认知度，但当前全球客户端仍以英语为主，因此排在第四。   |

### 实施顺序

第一阶段先上线：

```text
/en
/th
```

第二阶段增加：

```text
/ko
/ja
```

所有本地化页面都应设置独立的：

```text
title
description
H1
hreflang
canonical
Open Graph locale
```

建议的 `hreflang`：

```html
<link rel="alternate" hreflang="en" href="https://example.com/en/" />
<link rel="alternate" hreflang="th" href="https://example.com/th/" />
<link rel="alternate" hreflang="ko" href="https://example.com/ko/" />
<link rel="alternate" hreflang="ja" href="https://example.com/ja/" />
<link rel="alternate" hreflang="x-default" href="https://example.com/en/" />
```
